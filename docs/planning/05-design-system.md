# Design System (기초 디자인 시스템)

> NES 테트리스 스타일의 8bit 픽셀아트 디자인 가이드입니다.

---

## MVP 캡슐

| # | 항목 | 내용 |
|---|------|------|
| 1 | 목표 | 설치 없이 브라우저에서 바로 즐기는 추억의 고전 테트리스 |
| 2 | 페르소나 | 전 연령 가족 단위 (점심시간/짧은 휴식 시간에 플레이) |
| 3 | 핵심 기능 | FEAT-1: 테트리스 기본 플레이 |
| 4 | 성공 지표 (노스스타) | 평균 세션 지속 시간 10분 이상 |
| 5 | 입력 지표 | 일일 활성 사용자(DAU), 스테이지 10 클리어 비율 |
| 6 | 비기능 요구 | 60fps 유지, 반응형(PC+모바일), 정확한 조작감 |
| 7 | Out-of-scope | 소셜 로그인, 리더보드 공유, 멀티플레이 |
| 8 | Top 리스크 | 조작감이 기대에 미치지 못함 |
| 9 | 완화/실험 | NES 테트리스 프레임 테이블 참조하여 정확한 속도 구현 |
| 10 | 다음 단계 | API 계약 정의 및 테스트 케이스 작성 |

## 1. 디자인 철학

### 1.1 핵심 가치

| 가치 | 설명 | 구현 방법 |
|------|------|----------|
| 레트로 | NES 시대의 8bit 감성 재현 | 픽셀아트, 제한된 컬러 팔레트 |
| 집중 | 게임플레이에 집중할 수 있는 UI | 최소한의 장식, 명확한 정보 표시 |
| 향수 | 추억을 불러일으키는 경험 | 고전적인 사운드, 익숙한 레이아웃 |

### 1.2 참고 게임 (무드보드)

| 게임 | 참고할 점 | 참고하지 않을 점 |
|------|----------|-----------------|
| NES Tetris | 컬러 팔레트, UI 배치, 조작감 | - |
| 게임보이 Tetris | 간결한 인터페이스 | 모노크롬 색상 |
| Tetris 99 | 현대적 UI 요소 | 과한 이펙트 |

## 2. 컬러 팔레트

### 2.1 NES 스타일 컬러

NES의 제한된 팔레트를 모던하게 재해석합니다.

| 역할 | 컬러명 | Hex | 사용처 |
|------|------|-----|--------|
| **Background** | 네이비 블루 | `#0F0F23` | 전체 배경 |
| **Board BG** | 다크 그레이 | `#1A1A2E` | 게임 보드 배경 |
| **Grid** | 딤 그레이 | `#2D2D44` | 보드 그리드 라인 |
| **Text Primary** | 화이트 | `#FFFFFF` | 주요 텍스트 |
| **Text Secondary** | 라이트 그레이 | `#B8B8D0` | 보조 텍스트 |
| **Accent** | 골드 | `#FFD700` | 하이스코어, 강조 |

### 2.2 테트로미노 컬러

| 블록 | 모양 | Hex | 설명 |
|------|------|-----|------|
| **I** | 직선 | `#00F0F0` | 시안 |
| **O** | 정사각형 | `#F0F000` | 옐로우 |
| **T** | T자 | `#A000F0` | 퍼플 |
| **S** | S자 | `#00F000` | 그린 |
| **Z** | Z자 | `#F00000` | 레드 |
| **J** | J자 | `#0000F0` | 블루 |
| **L** | L자 | `#F0A000` | 오렌지 |

### 2.3 피드백 컬러

| 상태 | 컬러 | Hex | 사용처 |
|------|------|-----|--------|
| **Success** | 그린 | `#00F000` | 라인 클리어 |
| **Warning** | 옐로우 | `#F0F000` | 경고 (높은 스택) |
| **Error** | 레드 | `#F00000` | 게임 오버 |
| **Highlight** | 골드 | `#FFD700` | 테트리스(4줄), 신기록 |

## 5. 게임 UI 컴포넌트

### 5.1 게임 보드

```
┌─────────────────────────────────────┐
│ ┌─────────────────────────────────┐ │
│ │                                 │ │
│ │        10 x 20 그리드           │ │
│ │        블록 크기: 24px          │ │
│ │                                 │ │
│ │        총 크기: 240 x 480px     │ │
│ │                                 │ │
│ └─────────────────────────────────┘ │
│           테두리: 4px              │
└─────────────────────────────────────┘
```

| 속성 | 값 |
|------|-----|
| 보드 크기 | 10열 × 20행 |
| 블록 크기 | 24px × 24px (PC), 16px × 16px (모바일) |
| 테두리 | 4px solid #4A4A6A |
| 배경 | #1A1A2E |
| 그리드 | 1px solid #2D2D44 |

### 5.2 정보 패널

```
┌─────────────┐
│   SCORE     │  <- 라벨 (12px, #B8B8D0)
│   012500    │  <- 값 (16px, #FFFFFF)
├─────────────┤
│   LEVEL     │
│     05      │
├─────────────┤
│   LINES     │
│    045      │
└─────────────┘
```

### 5.3 다음 블록 미리보기

```
┌─────────────┐
│    NEXT     │
│ ┌─────────┐ │
│ │  ████   │ │  <- 4x4 그리드
│ │         │ │
│ └─────────┘ │
└─────────────┘
```

### 5.4 버튼

| 상태 | 스타일 |
|------|--------|
| 기본 | 배경 #2D2D44, 텍스트 #FFFFFF, 테두리 4px |
| 호버 | 배경 #4A4A6A |
| 포커스 | 테두리 #FFD700 |
| 비활성 | 50% 투명도 |

**크기:**
- Large: 높이 48px
- Medium: 높이 40px (기본)

## 6. 애니메이션

### 6.1 라인 클리어

```javascript
// 라인 클리어 애니메이션 (200ms)
1. 완성된 라인 깜빡임 (3회, 각 33ms)
2. 라인 사라짐 (페이드 아웃, 100ms)
3. 위 블록 떨어짐 (50ms)
```

### 6.2 레벨업

```javascript
// 레벨업 이펙트 (500ms)
1. "LEVEL UP!" 텍스트 표시 (중앙)
2. 텍스트 확대 + 페이드 아웃
3. 점수판 레벨 값 깜빡임
```

### 6.3 게임 오버

```javascript
// 게임 오버 (1000ms)
1. 보드 위에서 아래로 블록 채워지는 애니메이션
2. "GAME OVER" 텍스트 표시
3. 최종 점수 표시
```

## 7. 반응형 디자인

### 7.1 브레이크포인트

| 이름 | 범위 | 블록 크기 | 레이아웃 |
|------|------|----------|----------|
| Desktop | ≥1024px | 24px | 보드 좌측, 정보 우측 |
| Tablet | 768-1023px | 20px | 보드 중앙, 정보 상/하단 |
| Mobile | <768px | 16px | 세로 레이아웃, 터치 컨트롤 |

### 7.2 모바일 터치 컨트롤

```
┌───────────────────────────────────┐
│                                   │
│          [게임 보드]              │
│                                   │
├───────────────────────────────────┤
│                                   │
│    [◀]   [▼]   [▶]     [↻]      │
│    좌     드롭   우      회전     │
│                                   │
└───────────────────────────────────┘
```

| 버튼 | 크기 | 배치 |
|------|------|------|
| 방향 버튼 | 60px × 60px | 하단 좌측 |
| 회전 버튼 | 80px × 80px | 하단 우측 |
| 일시정지 | 40px × 40px | 상단 우측 |
